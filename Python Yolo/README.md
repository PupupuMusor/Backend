# TACO Trash Detection (YOLOv8)

Этот проект использует нейросеть YOLOv8 для распознавания мусора и классификации его по 6 категориям: Стекло, Металл, Органика, Бумага, Пластик, Прочее.

## Установка

1.  Установите Python 3.8+
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Использование

### 1. Скачивание датасета
Скачивает изображения с Flickr (многопоточно).
```bash
python download.py --threads 20
```

### 2. Подготовка данных
Конвертирует аннотации TACO в формат YOLO и применяет карту классов (6 категорий).
```bash
python convert_to_yolo.py
```

### 3. Обучение
#### Вариант А: Google Colab (Рекомендуется)
Загрузите файл `taco_colab.ipynb` в [Google Colab](https://colab.research.google.com/), выберите GPU T4 и запустите все ячейки. Скачайте полученный архив с моделью.

#### Вариант Б: Локально (Медленно)
```bash
python train_yolo.py
```

### 4. Запуск (Веб-камера)
Поместите файл обученной модели (например, `best.pt`) в папку проекта.
```bash
python webcam_yolo.py --model best.pt
```

### 5. Проверка данных
Просмотр того, как нейросеть "видит" обучающие данные (проверка рамок).
```bash
python inspect_yolo_data.py
```
